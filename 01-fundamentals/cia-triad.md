# CIA Triad

## Overview

The CIA Triad is one of the most important foundational concepts in cybersecurity.

CIA stands for:

- Confidentiality
- Integrity
- Availability

These three principles help organizations protect information and systems from unauthorized access, modification, and disruption.

---

# 1. Confidentiality

## Definition

Confidentiality means ensuring that information is only accessible to authorized individuals.

Sensitive information should not be disclosed to people who do not have permission to view it.

## Examples

- Student records should only be viewed by authorized school staff.
- Employee salary information should only be available to HR personnel.
- Customer passwords should never be publicly accessible.

## Threats to Confidentiality

- Data breaches
- Phishing attacks
- Weak passwords
- Unauthorized access
- Insider threats

## Security Controls

- Strong passwords
- Multi-Factor Authentication (MFA)
- Encryption
- Role-Based Access Control (RBAC)

---

# 2. Integrity

## Definition

Integrity ensures that information remains accurate, complete, and unaltered unless modified by authorized users.

Data should not be changed without permission.

## Examples

- Student grades should not be modified by unauthorized users.
- Financial records should remain accurate.
- Database entries should not be tampered with.

## Threats to Integrity

- Malware
- SQL Injection
- Unauthorized modifications
- Human error

## Security Controls

- Access controls
- Input validation
- Database constraints
- Audit logs
- Checksums and hashes

---

# 3. Availability

## Definition

Availability ensures that systems and data remain accessible when needed.

Authorized users should be able to access resources without unnecessary interruption.

## Examples

- School management systems should be available during registration periods.
- Websites should remain online for customers.
- Banking systems should operate reliably.

## Threats to Availability

- Denial of Service (DoS) attacks
- Hardware failures
- Power outages
- Ransomware
- Network failures

## Security Controls

- Backups
- Redundant systems
- Disaster recovery plans
- UPS power systems
- Monitoring and alerting

---

# Real-World Example

Consider a school management system.

## Confidentiality

Student records should only be visible to authorized staff.

## Integrity

Student marks should not be altered without authorization.

## Availability

The system should remain accessible during examinations and registration periods.

---

# Key Takeaway

The CIA Triad forms the foundation of cybersecurity.

Every security control can generally be linked to protecting:

- Confidentiality
- Integrity
- Availability

A secure system must protect all three.
